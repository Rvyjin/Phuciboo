HTML-Tags

Tag Description
<html> Defines the HTML document
<head> Metadata and page title
<title> Title in the browser tab
<body> Main content of the page
<h1> to <h2> Headings from largest (h1) to smallest (h6)
<p> Paragraph
<a href=""> Link to another page
<img src="" alt=""> Image with source (src) and alternative text (alt)
<ul>, <ol> Unordered (ul) and ordered (ol) list
<div> Block container for layout
<span> Inline container for styling
<strong> Bold text
<em> Italic text
<br> Line break
<hr> Horizontal line
<table>, <tr>, <td> Table with rows and cells
